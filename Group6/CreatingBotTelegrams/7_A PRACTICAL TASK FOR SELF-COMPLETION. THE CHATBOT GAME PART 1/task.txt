1. Создайте словарь users, наполненный минимум 3 людьми, следующего формата
{
1 : {‘name’ : ‘имя’,
	      ‘age’ : возраст(int)
	      ‘favorites’ : { 
				‘colors’ : [‘цвет1’, ‘цвет2’ ....],
				‘food’ : [‘еда1’, ‘еда2 ’....]
       	 }
	      }
}

2. Создайте глубокую копию словаря users в переменную users_copy
3. Поменяйте возраст второго человека на любое другое значение (int)  в словаре users_copy
4. Добавьте 3 человеку новый любимый цвет  в словаре users_copy
5. Выведите информацию о людях из двух словарей

Внимательно: на проверку необходимо отправить:
файл с кодом для бота БЕЗ токена,
